- definition of closed set / Cantor set
	- closed set = set that contains all its limit points (equivalently, complement is open)
	- Cantor set:
		- remove middle third from $[0,1]$, then repeat on remaining intervals
		- at each step remove an open interval → complement of union of open intervals
		- therefore Cantor set is closed (complement of an open set)

- closed sets / limit point (definition + picture hint)
	- definition of a limit point
		- author of book sometimes defines as "limit of a sequence from the set"
	- **Definition (Limit Point).** $a$ is a limit point of $A$ if every $V_{\epsilon}(a)$ contains points of $A$ other than $a$ itself.
		- **N.B.**: “other than $a$” is very important
		- intuition:
			- if you can always find other points of $A$ arbitrarily close to $a$, then $a$ is a limit point
			- otherwise $a$ is isolated
	- example: $A=\{1/n: n\in\mathbb{N}\}$
		- 0 is a limit point (even though $0\notin A$)
		- every other point $1/n$ is isolated:
			- gap between $\frac{1}{n}$ and $\frac{1}{n+1}$ equals $$ \frac{1}{n}-\frac{1}{n+1}=\frac{1}{n(n+1)}. $$
			- choose $\epsilon$ smaller than half that gap to isolate $\frac{1}{n}$
			- **N.B.**: [see pic](https://photos.janovitch.com/share/Zu0V6DdKxDV-OyPYF4fMFSxqVzdoD2rVTxDp8KREiUnIelPAlqeX4PKdHHWHwNbRzkY)

- 3.1.2 — is a set closed?
	- $A=\{1/n\}$:
		- 0 is a limit point not in $A$, so $A$ is not closed
	- $A=\{\sum_{k=1}^n \frac{1}{k^2}: n\in\mathbb{N}\}$ (partial sums)
		- partial sums converge to $\frac{\pi^2}{6}$
		- $\dfrac{\pi^2}{6}$ is a limit point (not in the set)
		- each finite partial sum is isolated
		- set is not closed
	- In contrast,
		- the harmonic series diverges; partial sums increase without bound
		- no finite accumulation point → partial sums are isolated

- Compactness
	- **Book's Definition (compactness, sequential).**
		- $K$ is compact if every sequence in $K$ has a converging subsequence $x_{n_k}$ whose limit is in $K$.
			- key upgrade vs BW: BW gives a convergent subsequence from boundedness, but does not promise the limit is in the set.
	- Most other books define compactness with **Heine–Borel** (in $\mathbb{R}$)
		- TFAE (The following are equivalent)
			1. $A$ is compact (sequential definition)
			2. $A$ is closed and bounded
				- closed = contains all limit points
				- bounded = sequence can’t “escape to infinity”
				- archetypal compact set is $[0,1]$
			3. Every open cover of $A$ has a finite subcover
				- open cover = collection of open sets whose union contains $A$
				- cover can be infinite or uncountable
				- compactness says: you can pick finitely many and still cover $A$
		- proof roadmap (professor emphasis: “need 4 proves”)
			- show (1) and (2) are equivalent
				- (1) ⇒ bounded + “limit stays in set” intuition
				- (2) ⇒ (1) uses BW + closedness to keep the limit inside
			- then show (3) ⇒ (2)
				- bounded part is not that hard (build a cover that forces boundedness)
				- the *interesting* part: (3) ⇒ closed
					- proof by contradiction
					- assume $A$ is not closed
						- then there exists a limit point $y$ of $A$ with $y \notin A$
						- i.e. there is a sequence $y_n \in A$ with $y_n \to y$ but $y \notin A$
					- crux construction (professor started but didn’t finish today):
						- for each $x \in A$, manufacture an $\epsilon$-neighborhood that avoids $y$
							- choose radius $\epsilon_x = \frac{1}{2}d(x,y)$ (half the distance to $y$)
							- then $V_{\epsilon_x}(x)$ cannot contain $y$
						- the collection $\{V_{\epsilon_x}(x) : x\in A\}$ is an open cover of $A$
						- **N.B.**: professor not up to proving it today
						- the contradiction idea:
							- any finite subcover has a minimum of those finitely many distances to $y$
							- then points of the sequence $y_n$ close enough to $y$ will not be covered
							- so no finite subcover exists
							- **N.B.**: [see pic](https://photos.janovitch.com/share/h8jaTrmfC4gfPC06qChocY1iZugpPY8XnyNvfZBZE_vUC93VPANHMI9yypzfTZBOVJU)
			- lastly show (2) ⇒ (3)
				- (closed + bounded) ⇒ every open cover has finite subcover
				- professor did not cover this today (likely will cover this theorem after exam)

- Is $\mathbb{Q}$ open or closed?
	- $\mathbb{Q}$ is neither open nor closed in $\mathbb{R}$
		- not closed: does not contain irrational limit points (e.g. sequences of rationals converging to $\pi$)
		- not open: any open ball around a rational contains irrationals

- 3.3.12 Every bounded infinite set has a limit point
	- proof idea (without invoking BW directly):
		- assume no limit point ⇒ every point is isolated
		- build an open cover by disjoint small neighborhoods around each point
		- since the set is infinite, no finite subcollection of those neighborhoods can cover the set ⇒ contradiction with compactness-type reasoning
		- therefore a bounded infinite set must have at least one limit point
