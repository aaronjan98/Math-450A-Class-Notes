- Monotone convergence via a convergent subsequence
    - Given: $x_n$ is increasing and has a convergent subsequence $(x_{n_k})$, then $x_n$ converges
    - idea: for an increasing sequence, the only missing ingredient is an upper bound
        - the convergent subsequence provides exactly that
    - the key is not to mess with the limit of $x_{n_k}$ — just use that it is bounded $(\dagger)$
    - proof $(\dagger)$
        - since $(x_{n_k})$ converges, it is bounded: there exists $M$ with $x_{n_k} \le M$ for all $k$
        - crux: for any $n$, choose $k$ so that $n_k \ge n$
        - because $(x_n)$ is increasing,
            - $x_n \le x_{n_k} \le M$
        - so the entire sequence is bounded above by $M$
        - by the Monotone Convergence Theorem, $(x_n)$ converges
    - exam-ready proof
        - let $(x_n)$ be increasing and suppose it has a convergent subsequence $(x_{n_k})$
        - because $(x_{n_k})$ converges, it is bounded above by some $M$
        - fix $n\in\mathbb{N}$ and choose $k$ such that $n_k \ge n$
        - since $(x_n)$ is increasing,
            - $x_n \le x_{n_k} \le M$
        - hence $(x_n)$ is bounded above
        - therefore, by the Monotone Convergence Theorem, $(x_n)$ converges

- **Abbott 4.4.7** (pg. 135) — Prove that $f(x) = \sqrt{x}$ is uniformly continuous on $[0, \infty)$
    - we want to make $|\sqrt{x} - \sqrt{y}|$ small
    - don't do the factorization thing
        - the identity $|\sqrt{x}-\sqrt{y}| = \dfrac{|x-y|}{\sqrt{x}+\sqrt{y}}$ is true
        - but the denominator $\sqrt{x}+\sqrt{y}$ depends on $x$ and $y$, so no uniform $\delta$ follows from it
            - near $x=y=0$, the denominator is small, meaning you'd need $|x-y|$ to be even smaller
            - the required $\delta$ would shrink to $0$ as $x,y\to 0$, so you cannot choose one $\delta$ for all pairs
    - use a crude estimate instead
        - start with the triangle inequality:
            - $|\sqrt{x}-\sqrt{y}| \le |\sqrt{x}| + |\sqrt{y}| = \sqrt{x} + \sqrt{y}$
        - now multiply both sides by $|\sqrt{x}-\sqrt{y}|$:
            - left side: $|\sqrt{x}-\sqrt{y}|^2$
            - right side: $(\sqrt{x}+\sqrt{y}) \cdot |\sqrt{x}-\sqrt{y}| = |(\sqrt{x}+\sqrt{y})(\sqrt{x}-\sqrt{y})| = |x-y|$ (difference of squares)
        - this gives the key estimate — the handle $|x-y|$ appears:
            - $|\sqrt{x}-\sqrt{y}|^2 \le |x-y|$
        - taking square roots:
            - $|\sqrt{x}-\sqrt{y}| \le \sqrt{|x-y|}$
        - we've related what we wanted to make small with the handle $|x-y|$
    - now choose $\delta$
        - given $\epsilon>0$, take $\delta=\epsilon^2$
        - if $|x-y|<\delta$, then
            - $|\sqrt{x}-\sqrt{y}| \le \sqrt{|x-y|}<\sqrt{\delta}=\epsilon$
        - therefore $\sqrt{x}$ is uniformly continuous on $[0,\infty)$
    - a crude estimate like this is not always available — the trick works here because the triangle inequality is tight enough once you multiply through
    - exam-ready proof
        - let $\epsilon>0$ be given and choose $\delta=\epsilon^2$
        - if $x,y\in[0,\infty)$ and $|x-y|<\delta$, then
            - $|\sqrt{x}-\sqrt{y}|^2 \le |x-y|<\delta=\epsilon^2$
        - taking square roots gives $|\sqrt{x}-\sqrt{y}|<\epsilon$
        - hence $\sqrt{x}$ is uniformly continuous on $[0,\infty)$

- **Abbott 4.4.8(a)** (pg. 134) — Can a continuous function on $[0,1]$ have range $(0,1)$?
    - if $K$ is compact and $f:K\to\mathbb{R}$ is continuous, then $f(K)$ is compact
        - in $\mathbb{R}$, compact means closed and bounded
    - the answer is no — such a function cannot exist
    - proof
        - $[0,1]$ is compact, so $f([0,1])$ must be compact
        - but $(0,1)$ is not compact since it is not closed
        - contradiction
    - another way to remember this
        - a continuous function on a compact interval attains its maximum and minimum
        - a function with range exactly $(0,1)$ would miss both endpoint values, which is impossible on $[0,1]$

- **Abbott 4.4.8(b)(c)** (pg. 134) — examples: compactness of the domain matters
    - if the domain is not compact, the compact-range conclusion can fail
    - for a half-open domain
        - example: $f(x)=x$ on $(0,1]$ has range $(0,1]$, which is not compact
    - for an open domain
        - it is possible to have a continuous function on $(0,1)$ with range $[0,1]$
        - standard example: $f(x)=\dfrac{1+\sin(1/x)}{2}$ on $(0,1)$
        - why this works
            - $\sin(1/x)$ takes the values $\pm 1$ infinitely often as $x\to 0^+$
                - $1/x = \pi/2 + k\pi$ for $k=0,1,2,\ldots$ gives alternating $\pm 1$, with those $x$ values in $(0,1)$ for large $k$
            - after shifting and scaling, the range becomes exactly $[0,1]$
    - moral: compactness of the domain is the key hypothesis — once it is gone, many of the nice range restrictions disappear

- derivatives need not be continuous
    - standard example: $g(x)=x^2\sin(1/x)$ for $x\ne 0$, $g(0)=0$
    - differentiability at $0$:
        - $\displaystyle g'(0)=\lim_{x\to 0}\frac{g(x)}{x} = \lim_{x\to 0}x\sin(1/x)=0$
        - since $|x\sin(1/x)|\le |x|$, the limit is $0$ by squeeze
    - for $x\ne 0$:
        - $g'(x)=2x\sin(1/x)-\cos(1/x)$
        - the $\cos(1/x)$ term oscillates as $x\to 0$, so $g'$ is not continuous at $0$
    - so $g$ is differentiable everywhere yet $g'$ is not continuous
    - what it means for a function to be a derivative
        - $h$ is a derivative if there exists $F$ with $F'=h$
        - example: $\dfrac{x^2}{2}$ is a derivative because it equals $\left(\dfrac{x^3}{6}\right)'$
        - that example happens to have a continuous derivative, but that is not true in general

- **Abbott 4.5.7** (pg. 140) — the fixed point argument uses IVT
    - to show a function has a fixed point on $[a,b]$, define $h(x)=f(x)-x$
    - if $h(a)\ge 0$ and $h(b)\le 0$ (or vice versa), then by IVT there exists $c\in[a,b]$ with $h(c)=0$
    - that means $f(c)=c$
    - geometrically: you're showing the graph of $f$ must cross the diagonal $y=x$

- **Abbott 4.2.2(b)** (pg. 120) — finding the largest $\delta$: $\lim_{x \to 4} \sqrt{x} = 2$, $\epsilon = 1$
    - we want the largest $\delta$ such that $|x-4|<\delta \implies |\sqrt{x}-2|<1$
    - solve the output condition $|\sqrt{x}-2|<1$ directly:
        - $-1<\sqrt{x}-2<1 \implies 1<\sqrt{x}<3 \implies 1<x<9$
    - draw a number line: we need $x \in (1,9)$
        - distance from $4$ to left endpoint $1$ is $3$
        - distance from $4$ to right endpoint $9$ is $5$
        - the smaller controls the answer
    - so the largest possible $\delta$ is $\mathbf{3}$
        - if $\delta>3$, then $x\le 1$ is allowed, but then $|\sqrt{x}-2|\ge 1$
