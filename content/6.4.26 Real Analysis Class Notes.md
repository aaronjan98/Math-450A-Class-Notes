- (pg. 128) - Contraction Theorem
  - $|f(x) - f(y)| \leq c |x-y| \qquad 0 < c < 1$
    - such a function is called a contraction
    - a lot of approximation techniques use the contraction method to find fixed points
      - the idea is to start somewhere and iterate:
        - $x_{n+1} = f(x_n)$
      - the contraction condition forces successive iterates to get closer and closer
      - this is why fixed points can often be found by iteration
    - a contraction has at most one fixed point
      - if $p$ and $q$ are fixed points, then
        - $|p-q| = |f(p)-f(q)| \le c|p-q|$
      - since $0<c<1$, this implies $|p-q|=0$
      - so $p=q$
    - on a complete metric space, a contraction has a unique fixed point and the iterates converge to it
      - in our setting, this is the basic fixed-point picture to keep in mind
    - e.g. Newton's Method
      - is Newton's method an iterative method of the contraction type?
      - if we're trying to find the root of a differentiable function, then we get the tangent line starting at some iterate $x_1$
      - we find the tangent line to the graph at $(x_1,f(x_1))$
      - $x_2$ is where that tangent line crosses the $x$-axis
      - then repeat
      - Newton's method is iterative, but it is not automatically a contraction on all of $\mathbb{R}$
        - in some regions it behaves like a contraction
        - in other cases it can fail badly
      - so the general contraction theorem and Newton's method are related in spirit, but not identical

- **Exercise 4.2.5** - Use Definition 4.2.1 to prove:
  - (d) $\displaystyle \lim_{x\to 3} 1/x = 1/3$
    - **Proof.** Inversions in such problems become counterintuitive
    - We first say that $\epsilon$ is arbitrary, so
      - given $\epsilon > 0$
      - find $\delta > 0 \ni \left\lvert  x - 3  \right\rvert < \delta \longrightarrow \left| \frac{1}{x} - \frac{1}{3} \right| < \epsilon$
    - then
      - $\left| \frac{1}{x} - \frac{1}{3} \right| = \left| \frac{3-x}{3x} \right| = \frac{|3-x|}{3|x|}$
      - we want the denominator bounded away from $0$
      - if $x$ is close to $0$, we get a problem, so we trap $x$ near $3$
    - assume $\delta < 1$
      - then $|x-3|<1$ implies $2<x<4$
      - in particular, $|x|>2$
      - so
        - $\frac{|3-x|}{3|x|} \le \frac{|3-x|}{6}$
    - choose $\delta = \min(1, 6\epsilon)$
    - then if $|x-3|<\delta$, we get
      - $\left| \frac{1}{x} - \frac{1}{3} \right| = \frac{|3-x|}{3|x|} \le \frac{|3-x|}{6} < \frac{\delta}{6} \le \epsilon$
    - thus $\displaystyle \lim_{x\to 3} \frac{1}{x} = \frac{1}{3}$
    - **N.B.**: theorem-like feel because of the quotient, but this can be done directly from the definition

- **Fixed point problem — Exercise 4.3.11(c)** *(professor's favorite; most likely on the exam)*
  - Context: $f:\mathbb{R}\to\mathbb{R}$ is a contraction — it satisfies $|f(x)-f(y)|\le c|x-y|$ for all $x,y$, where $0<c<1$. Parts (a) and (b) establish that $f$ is continuous and that if you pick any starting point $y_1$ and keep applying $f$, the resulting sequence of iterates $y_1,\ y_2=f(y_1),\ y_3=f(y_2),\dots$ converges to some limit $y$. Part (c) asks you to show that limit is actually a fixed point — a point the function sends back to itself.
  - **(c) Prove $y = \lim y_n$ is a fixed point of $f$ (i.e. $f(y)=y$), and that it is the only one.**
  - proof that $y$ is a fixed point
    - since $y_n\to y$ and $f$ is continuous (from part (a)):
      - $f(y_n)\to f(y)$
    - but by definition of the iterates, $f(y_n)=y_{n+1}$
    - and since $y_n\to y$, the shifted sequence $y_{n+1}\to y$ as well
    - so both sides of $f(y_n)=y_{n+1}$ converge to the same limit:
      - $f(y)=y$ $\square$
  - proof of uniqueness
    - suppose $p$ and $q$ are both fixed points, i.e. $f(p)=p$ and $f(q)=q$
    - apply the contraction condition:
      - $|p-q| = |f(p)-f(q)| \le c|p-q|$
    - rearranging: $(1-c)|p-q| \le 0$
    - since $0<c<1$, we have $1-c>0$, so this forces $|p-q|=0$, i.e. $p=q$ $\square$

- given $$ \left. \begin{aligned} f^{\text{cont}} \text{ on } [0,1] \\ R_f \subseteq [0, 1] \end{aligned} \right\} f \text{ has a fixed point in } [0,1] $$
  - this means: given $f:[0,1]\to[0,1]$ continuous (range *contained in* $[0,1]$), prove $\exists\, c\in[0,1]$ with $f(c)=c$
  - **N.B.**: the key is that the domain and range are the same interval — the graph of $f$ is forced to cross the diagonal $y=x$
  - going to use IVT here to formalize that crossing argument
  - *(aside — for the final, not this midterm: derivatives have the intermediate value property even when not continuous; this is Darboux's theorem. A classic example is a function that oscillates faster and faster near $0$, squeezed under $y=x$.)*
  - How to use the IVT
    - the crux is creating a function that satisfies the sign change condition
    - the trick is to take
      - $h(x) = f(x) - x$
    - since $f$ and $x\mapsto x$ are continuous, $h$ is continuous on $[0,1]$
    - now check the endpoints
      - $h(0)=f(0)-0=f(0)$
      - because $f([0,1])\subseteq [0,1]$, we have $f(0)\ge 0$
      - so $h(0)\ge 0$
      - if $h(0)=0$, then $f(0)=0$ and we are done
      - next,
        - $h(1)=f(1)-1$
      - because $f([0,1])\subseteq [0,1]$, we have $f(1)\le 1$
      - so $h(1)\le 0$
      - if $h(1)=0$, then $f(1)=1$ and we are done
    - therefore, either we already have a fixed point at an endpoint, or else
      - $h(0)>0$ and $h(1)<0$
    - By IVT, $\exists c \in (0, 1)$ with $h(c) = 0$
      - if $h(c) = 0 \Rightarrow f(c) - c = 0 \Rightarrow f(c)=c$
      - so $c$ is a fixed point of $f$
  - to [complete the picture](https://photos.janovitch.com/share/U5y52ZoByj9Dw6OQu2i95gVnxTujIcN0qHZV6watv07yTVx5lgdS4p2BOlhDSpstusU): draw the unit square $[0,1]\times[0,1]$ with the diagonal $y=x$ cutting through it. Any continuous $f:[0,1]\to[0,1]$ starts its graph at or above $(0,0)$ (since $f(0)\ge 0$ places it above the diagonal at $x=0$) and ends at or below $(1,1)$ (since $f(1)\le 1$ places it below the diagonal at $x=1$). A curve that starts above $y=x$ and ends below it must cross the diagonal — that crossing is the fixed point.
    - the same sign-change logic appears in 4.3.8(a): if $g\ge 0$ for $x<1$ and $g$ is continuous, then $g(1)\ge 0$ — both use continuity to "carry" an inequality across to a limit point
    - what does the closed interval requirement play in this problem
      - the theorem is stated on $[0,1]$, and that guarantees the endpoint values $f(0)$ and $f(1)$ are available
      - IVT itself works on intervals, but here the closed interval setup is exactly what makes the endpoint argument clean
    - thinking of a counterexample for open sets
      - if the domain were $(0,1)$ instead, you could avoid a fixed point
      - for example, $f(x)=\frac{x+1}{2}$ maps $(0,1)$ into $(0,1)$, is continuous, and has no fixed point in $(0,1)$

- **Exercise 4.4.7** — Triangle Inequality proof
  - Show that $g(x) = \sqrt{ x }$ is uniformly continuous (shorthand: UC) on $[0, \infty)$
    - the $\delta$ has to work for all pairs of points simultaneously, including near $0$ where $\sqrt{x}$ changes fastest
    - the crux is the very first step
      - we want to make $|\sqrt{ x } - \sqrt{ y }|$ small
      - to show UC, we need: for every $\epsilon>0$, one $\delta>0$ such that $|g(x) - g(y)| < \epsilon$ whenever $|x-y| < \delta$, for *all* $x,y\ge 0$
        - *(note: $\delta$ depends only on $\epsilon$, not on the specific points $x$ and $y$ — that's what makes it uniform)*
        - we can assume $x \neq y$ since the case $x=y$ is trivial
      - the clean estimate is
        - $|\sqrt{ x } - \sqrt{ y }| \leq \sqrt{|x-y|}$
      - one way to derive it is:
        - assume without loss of generality that $x\ge y$
        - then
          - $|\sqrt{x}-\sqrt{y}|=\sqrt{x}-\sqrt{y}$
        - square both sides
          - $(\sqrt{x}-\sqrt{y})^2 = x+y-2\sqrt{xy}$
          - since $x\ge y\ge 0$, we have $\sqrt{y}\le\sqrt{x}$, so $2\sqrt{xy}=2\sqrt{x}\cdot\sqrt{y}\ge 2y$, which gives $x+y-2\sqrt{xy}\le x+y-2y = x-y$
          - and since $x\ge y$, $x-y=|x-y|$, so: $(\sqrt{x}-\sqrt{y})^2\le |x-y|$
        - then take square roots
          - $|\sqrt{x}-\sqrt{y}| \le \sqrt{|x-y|}$
      - this is the key inequality
        - **N.B.**: this is the part to get right on the exam
      - so how small do we have to make $\delta$ to make this smaller than $\epsilon$?
        - choose $\delta = \epsilon^{2}$
      - then if $|x-y|<\delta$, we have
        - $|\sqrt{x}-\sqrt{y}| \le \sqrt{|x-y|}<\sqrt{\delta}=\epsilon$
      - hence $\sqrt{x}$ is uniformly continuous on $[0,\infty)$

- What is the negation of uniformly continuous?
  - If $f: A \to \mathbb{R}$ is NOT UC, then
    - $\exists \epsilon_{0} > 0 \ni \forall \delta > 0$, we can find $x, y \in A$ such that
      - $|x-y| < \delta$, yet $|f(x) - f(y)| \geq \epsilon_{0}$
    - if it was uniformly continuous, then for each $\epsilon$ you could find a $\delta$ such that the output values are closer than $\epsilon$
    - otherwise, there's an $\epsilon_{0}$ that you can't accommodate for any $\delta$
  - the book uses this negation to prove...
    - if $f$ is continuous and defined on a compact set $K$, then the function is UC on $K$
      - this is the Heine-Cantor theorem
      - you prove this by contradiction
      - **N.B.**: will go over this proof after the exam
  - If $$\left.\begin{aligned} f^{\text{cont}} \\ x_{n} \rightarrow x \end{aligned} \right\} \rightarrow f(x_{n}) \to f(x) $$
    - this is the sequential characterization of continuity
    - **N.B.**: property/theorem, so on the test we can say because $f$ is continuous, this implies $f(x_n)\to f(x)$

- if $f$ is continuous, then $f$ takes compact sets to compact sets
  - **N.B.**: didn't go over this, but should know it
  - trying to think of an example..
    - if $f$ is continuous on a closed interval $[a,b]$, then $f([a,b])$ is compact
    - in $\mathbb{R}$, compact means closed and bounded
    - **N.B.**: review closed vs compact
  - sequential way to think about compactness
    - a set $K$ is compact if every sequence in $K$ has a convergent subsequence whose limit is still in $K$
  - to show $f(K)$ is compact
    - take a sequence $(y_n)$ in $f(K)$
    - for each $n$, choose $x_n\in K$ such that $f(x_n)=y_n$
      - want to get the corresponding sequence of preimages
    - since $K$ is compact, $(x_n)$ has a convergent subsequence $x_{n_k}\to x\in K$
    - because $f$ is continuous,
      - $f(x_{n_k})\to f(x)$
    - but $f(x_{n_k})=y_{n_k}$
    - so $(y_{n_k})$ is a convergent subsequence of $(y_n)$ whose limit is in $f(K)$
    - therefore $f(K)$ is compact
